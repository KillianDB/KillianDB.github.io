HTML

HTML, which stands for HyperText Markup Language, is a markup language used to structure and format web pages. It defines the basic elements and content of a webpage, allowing browsers to interpret and display the information to users.

CSS

CSS, which stands for Cascading Style Sheets, is a language used to style and format web pages. It controls the appearance of HTML elements, including fonts, colors, spacing, and layout.

MySQL

MySQL is an open-source, fast, and widely used relational database management system that supports SQL language.

JavaScript

JavaScript is a versatile language for interactive web development used both for frontend and backend propouse. It enables dynamic elements, user interactions, and data processing.

ReactJS

ReactJS is an open-source JavaScript library for building user interfaces, particularly single-page applications (SPAs). It allows developers to create interactive and dynamic components that efficiently update and render data as it changes, reducing the need for full-page refreshes.

Python

Python is a high-level and versatile programming language focused on simplicity and efficiency, making it easier to work with.

Golang

Go, aka Golang, is an open-source, efficient and statically-typed programming language developed by Google. It is well-suited for developing scalable and high-performance applications, as it has a strong emphasis on concurrent code execution and fast compilation.

Git

Git is a distributed version control system that enables developers to track changes in their codebase, collaborate with others, and manage multiple versions of their projects efficiently.
